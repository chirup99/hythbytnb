# 1️⃣ Latest F&O Charges (NSE)

### Brokerage

* Futures: **0.03% or ₹20/order (whichever lower)**
* Options: **₹20 per executed order** ([Zerodha][1])

So normally:

* Buy order = ₹20
* Sell order = ₹20

---

# 2️⃣ Government & Exchange Charges

| Charge               | Futures                                | Options                  |
| -------------------- | -------------------------------------- | ------------------------ |
| STT                  | **0.02% on sell value**                | **0.1% on sell premium** |
| Exchange transaction | **0.00173%**                           | **0.03503% (premium)**   |
| SEBI charges         | ₹10 per crore                          | ₹10 per crore            |
| Stamp duty           | **0.002% buy side**                    | **0.003% buy side**      |
| GST                  | **18% on brokerage + exchange + SEBI** |                          |
| IPFT                 | 0.0001%                                | 0.0005%                  |

([Share India Securities Ltd][2])

---

# 3️⃣ Formula to Calculate Charges

### Step 1

Calculate **Turnover**

For futures

```
Turnover = (Buy price + Sell price) × Lot size
```

For options

```
Premium turnover = (Buy premium + Sell premium) × Lot size
```

---

### Step 2

Calculate each charge

**Exchange charge**

```
Turnover × 0.00173%
```

**SEBI charge**

```
Turnover × 0.0001%
```

**Stamp duty**

```
Buy value × 0.002% (futures)
Buy premium × 0.003% (options)
```

**STT**

```
Sell value × 0.02% (futures)
Sell premium × 0.1% (options)
```

**GST**

```
18% × (Brokerage + Exchange + SEBI + IPFT)
```

# 4️⃣ Example (NIFTY Option Trade)

Example trade

Buy: **₹100 premium**
Sell: **₹120 premium**
Lot size: **50**

### Step 1

Premium turnover

```
(100 + 120) × 50
= ₹11,000
```

---

### Step 2 Charges

Brokerage

```
20 + 20 = ₹40
```

Exchange charge

```
11000 × 0.03503%
≈ ₹3.85
```

SEBI charge

```
11000 × 0.0001%
≈ ₹0.01
```

Stamp duty

```
100 × 50 × 0.003%
≈ ₹0.15
```

STT

```
120 × 50 × 0.1%
= ₹6
```

GST

```
18% × (40 + 3.85 + 0.01)
≈ ₹7.90
```

---

✅ **Total charges**

```
≈ ₹58
```
